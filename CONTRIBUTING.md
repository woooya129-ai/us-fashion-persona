## DCO Sign-Off

All contributions must include a Developer Certificate of Origin 1.1 sign-off
in each commit message:

```text
Signed-off-by: Your Name <your.email@example.com>
```

You can add it with:

```powershell
git commit -s
```

The DCO sign-off means you certify that you have the right to submit the
contribution under the project's license.

This follows the lightweight DCO sign-off model used by large open-source
projects. It is not a copyright assignment.
