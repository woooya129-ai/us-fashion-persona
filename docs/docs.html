이 도구가 좁히는 범위

완전한 SaaS도, 실제 소비자 조사 대체도 아닙니다. 패션 컨셉을 합성 패널에 빠르게 확인하는 보조도구입니다.

1

US fashion 전용 질문 설계

일반 시장조사 도구가 아니라 브랜드 컨셉, 가격, 착용 상황, 스타일 톤, 메시지 반응을
묻는 패션 전용 screener.

2

US fashion 실무자용 local app

SaaS가 아니라 local Streamlit GUI입니다.
API Key와 결과는 사용자 실행 환경에 남습니다.

3

제품 카드 → 균형형 패널 평가 → 수정 제안 후보 → CSV/Markdown 리포트

입력에서 리포트까지 한 흐름입니다. v0.5.3은 USA 데이터셋, BLS/Census/Federal Reserve 집계 통계, LLM API 가격 추정을 함께 정리합니다.
리포트에서는 concern 문장을 패션 리스크 7분류로 집계하고, deterministic rule 기반 수정 제안 후보와 대표 페르소나 카드로 정리합니다.

4

가격·소득·자산 맥락값

BLS 의류 지출, Census 중위 가구소득, Federal Reserve SCF 순자산 기준을 함께 표시합니다.
개인별 구매력이나 구매 의향 예측은 아닙니다.

5

실제 소비자 조사 대체 아님 / A-B 상대 비교 중심

합성 패널 기준이므로 절대 수치 단정은 피해야 합니다. 컨셉 A·B 상대 비교에 적합합니다.

포함과 한계

로컬 베타 범위는 좁게 설정되어 있습니다. 확장 영역은 후속 작업으로 분리합니다.

지금 할 수 있는 것 (로컬 앱에서)

  • 제품 카드 입력
    카테고리, 가격, 핏, 소재, 컬러, 시즌, 착용 상황, 스타일 톤, 타깃 가설, 브랜드 메시지
  • 모델 선택 + 토큰/비용/시간 사전 추정
  • 합성 패널 응답을 취향·관심·망설임·리스크 등 7개 관점으로 균형 있게 평가(리포트 집계는 concern·리스크 카테고리 포함)
  • deterministic rule 기반 수정 제안 후보 자동 생성
  • 대표 페르소나 카드와 CSV / Markdown 리포트 다운로드
  • 본인 API Key를 로컬 secret 경로에서 로드

이 페이지/저장소가 다루지 않는 것

  • 이 정적 페이지에서의 평가 실행 (도구는 로컬 앱 탭에서)
  • 호스팅 SaaS / 결제 / 계정 / 크레딧
  • 실제 소비자 조사를 대체하는 결론
  • 비즈니스 성과나 시장 점유 단정
  • 합성 패널을 전체 소비자로 일반화
  • 실제 매출/판매 결과 보장
  • 대기업 납품형 보안·계약 문서

안내. 본 페이지는 설명용 정적 문서이며, 평가 실행은 로컬 앱 탭에서만 진행합니다. 공개 저장소는 호스팅 평가 서비스가 아닙니다. 본 도구는 합성 페르소나와 LLM 기반의 사전 가설 분석 보조도구로, 합성 패널 반응 분포를 보여줄 뿐 실제 소비자 반응이나 비즈니스 판단을 단정하지 않습니다. 최종 의사결정은 사람의 판단과 정식 조사·자문을 함께 사용해야 합니다.
